    Frequently Asked Questions

    Is PixelForge safe for confidential corporate documents?

    Yes, absolutely. Unlike traditional online converters, PixelForge functions 100% client-side via WebWorkers and WebAssembly inside an isolated browser sandbox. Your financial data, military text maps, or private photos are never uploaded to foreign servers.

    What is the maximum file size allocation?

    PixelForge features a hardware-optimized resource guardian that safely limits local conversions up to 500MB per file. This rule prevents browser memory leakage and Out-of-Memory crashes during intense mathematical conversions.

    Does batch conversion preserve formatting?

    Yes. When compiling images (JPG/HEIC to PDF) or parsing files (PDF to EPUB), our typographic engine maintains responsive grids and sequential styling layout trees using standard IDPF formatting entirely in local RAM.
